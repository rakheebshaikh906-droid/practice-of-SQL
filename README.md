# SQL Practice

A collection of SQL queries, exercises, and database practice problems designed to strengthen SQL skills from beginner to advanced levels.

## Overview

This repository contains various SQL concepts, queries, and hands-on practice exercises that help in understanding database management and data retrieval techniques. It serves as a learning resource for students, aspiring data analysts, and developers preparing for interviews.

## Topics Covered

* Basic SQL Queries
* SELECT Statements
* WHERE Clause
* ORDER BY
* GROUP BY
* HAVING Clause
* Aggregate Functions

  * COUNT()
  * SUM()
  * AVG()
  * MIN()
  * MAX()
* Joins

  * INNER JOIN
  * LEFT JOIN
  * RIGHT JOIN
  * FULL JOIN
* Subqueries
* Views
* Constraints
* Database Normalization
* SQL Interview Questions
* Real-world Query Practice


## Learning Objectives

* Understand relational database concepts.
* Write efficient SQL queries.
* Perform data filtering and sorting.
* Work with multiple tables using joins.
* Analyze and summarize data using aggregate functions.
* Solve real-world database problems.
* Prepare for SQL interviews and assessments.
